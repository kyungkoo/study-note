### print
Java에서 System 출력을 위해서는
```Java
System.out.println("This is Java...");
```

Swift 에서는 `print`로 동일한 결과를 얻는다.
```Swift
print("This is Swift2...")
```

### for Loop
Java의 for loop는 다음과 같다.
```Java
for (int i = 0; i < 10; i++) {
  // do something
}
```

Swift 에서는 다음과 같은 방식으로 동일한 결과를 얻는다.
```Swift
for 0..<10 {
  // do something
}
```
